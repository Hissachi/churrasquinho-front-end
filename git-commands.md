# Comandos úteis do Git

Este guia apresenta os principais comandos do Git utilizados no desenvolvimento.

---

## 1. git clone

Faz uma cópia de um repositório remoto para sua máquina local.

clone do projeto:
git clone https://github.com/Hissachi/churrasquinho-front-end.git

---

## 2. git status

Mostra o estado atual do repositório.

git status

---

## 3. git add

Adiciona arquivos para a área de preparação (staging area).
Antes de criar um commit, os arquivos precisam ser adicionados.

Exemplo:
*Adicionar um arquivo específico*
git add arquivo.js

*Adicionar todos os arquivos*
git add .

---

## 5. git push

Envia os commits locais para o repositório remoto (GitHub).

git push

---

## 6. git pull

Atualiza seu repositório local com as alterações do repositório remoto.

git pull

---

## 7. git branch

Lista todas as branches existentes no projeto:
git branch

Cria uma nova branch:
git branch nome-da-branch

---

## 8. git checkout

Troca para outra branch:
git checkout nome-da-branch

Cria e troca para uma branch:
git checkout -b nova-branch

---
